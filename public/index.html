Getting Started

1
Install wheat (the research engine)
Wheat is the core tool. Everything starts with a research sprint.
npx @grainulation/wheat init
2
Run the ecosystem doctor
Check what's installed and what's available.
npx grainulation doctor
3
Add tools as you need them
Each tool has a purpose. Start small, add as your research grows.
npx grainulation setup
4
Slow down and trust the process
Research takes time. Claims build evidence. Evidence builds decisions.
grainulation ecosystem · slow down and trust the process
